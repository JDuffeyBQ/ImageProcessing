<h3>ImageProcessing Plugin</h3>

<p>This plugin contains filters that implements various image processing algorithms. Each of the algorithms currently wrap a specific filter from the Insight Segmentation Toolkit (www.itk.org). This plugin thus depends directly on the ITK libraries being available within the system or redistributable package.</p>
